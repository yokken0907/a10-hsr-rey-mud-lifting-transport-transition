# Safe Wording Table / 安全表現表

## Purpose / 目的

This table provides recommended wording for GitHub, Jxiv, INPIT, public-agency, and researcher-facing explanations.

この表は、GitHub、Jxiv、INPIT、公的機関、研究者向け説明で過剰主張を避けるための表現集である。

## Core boundary

### English

HSR v3.x is a reduced-order, pre-engineering transport-transition and safety-diagnostic framework for ultra-deep REY mud lifting. It is not a sea-trial-validated mining system, not a final riser design, not certified offshore equipment, and not a commercial-deployment package.

### Japanese

HSR v3.xは、超深海REY泥揚鉱を対象とするreduced-orderの前工学的輸送遷移・安全診断フレームワークであり、実海域検証済み採鉱装置、最終ライザー設計、安全認証済み技術、または商用化準備完了技術を主張するものではない。

## Recommended wording

| Context | Safer English wording | Safer Japanese wording |
|---|---|---|
| Overall identity | reduced-order pre-engineering framework | reduced-orderの前工学的フレームワーク |
| System description | transport-transition surrogate | 輸送遷移サロゲート |
| Practical value | validation-oriented screening framework | 検証に接続する候補選別フレームワーク |
| Diagnostic claim | candidate diagnostic trigger | 候補診断トリガ |
| Sustained result | sustained in the tested reduced-order model | 検証対象の縮約モデル内で持続輸送を維持 |
| Critical velocity | estimated transition region in the current model | 現在のモデル内で推定された遷移領域 |
| Deposition | deposition remains a next-stage constraint | 堆積は次段階の制約として残る |
| Industry relevance | may support early-stage review | 初期段階の検討を支援し得る |
| Validation | requires further CFD/FSI, experimental, and field review | CFD/FSI、実験、実海域レビューが必要 |
| Repository status | paper companion and review archive | 論文補助・レビュー用アーカイブ |

## Wording to avoid

| Avoid | Reason |
|---|---|
| field-validated mining system | No sea-trial validation is included |
| final riser design | No final hardware design is included |
| certified offshore equipment | No safety certification is included |
| commercial-ready | No deployment or permitting evidence is included |
| completed deposition management | Deposition remains unresolved |
| operational alarm rule | The diagnostic is only a candidate trigger |
| practically proven | Evidence is reduced-order and pre-engineering |
| safe operation guaranteed | Safety certification and field validation are absent |

## Japanese wording to avoid

| 避ける表現 | 理由 |
|---|---|
| 実海域検証済み採鉱装置 | 実海域試験の証拠がない |
| 最終ライザー設計 | 最終ハードウェア設計ではない |
| 安全認証済み技術 | 安全認証を受けていない |
| 商用化準備完了 | 商用展開・許認可・操業実績がない |
| 堆積管理を完成 | 長時間堆積管理は未解決 |
| 運用アラーム規則 | 候補診断量に留まる |
| 実用性を証明 | reduced-orderの前工学的結果に留まる |
| 安全運用を保証 | 安全認証・実海域検証がない |

## Compact explanation

### English

> HSR v3.x is best read as a validation-oriented pre-engineering framework. It organizes transport-transition risk, blackout/coastdown collapse, deposition constraints, and candidate diagnostics before higher-fidelity simulation, experiment, and field validation.

### Japanese

> HSR v3.xは、より高忠実度のシミュレーション、実験、実海域検証へ進む前に、輸送遷移リスク、blackout/coastdown collapse、堆積制約、候補診断量を整理する検証志向の前工学的フレームワークとして読むのが安全である。
