# INPIT・公的機関向け1枚説明資料：HSR v3.x

## 技術名

**Hybrid Smart Riser / HSR v3.x**  
超深海REY泥揚鉱のための輸送遷移・ブラックアウト診断・堆積制約フレームワーク

## 一文要約

HSR v3.xは、超深海REY泥揚鉱を対象とするreduced-orderの前工学的輸送遷移・安全診断フレームワークであり、実海域検証済み採鉱装置、最終ライザー設計、安全認証済み技術、または商用化準備完了技術を主張するものではない。

## 背景

レアアース資源の安定確保は、将来的に重要な産業・安全保障上の課題になり得る。  
一方で、超深海REY泥の揚鉱では、単に泥を吸い上げるだけではなく、以下のような輸送安全上の問題がある。

- 流速不足による輸送相の崩壊
- blackout / coastdown 時の急激な失速
- 長時間運転による堆積成長
- 停止後の再起動困難化
- 実海域試験前に危険条件をどう除外するか
- どのセンサ量を見れば危険遷移を検出できるか

## HSR v3.x が扱う内容

HSR v3.xは、reduced-order modelを用いて以下を整理する。

- sustained transport：通常運転下で輸送が維持される領域
- collapse sector：blackout/coastdownなどの障害で崩壊しやすい領域
- deposition-limited horizon：長時間運転で堆積が支配制約になる領域
- residual pressure-gradient derivative：blackout診断の候補トリガ
- next validation plan：実鉱泥・CFD/FSI・センサ検証・長時間堆積検証へ進むための計画
