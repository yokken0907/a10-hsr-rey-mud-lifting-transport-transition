# README Insert Blocks

This file contains optional text blocks that can be inserted into `README.md` and `README_ja.md`.

## Suggested insertion for `README.md`

Place near the top, after the title and manuscript identity.

```markdown
## Start here

This repository is not a field-ready mining system.  
It is a reduced-order pre-engineering transport-transition and safety-diagnostic framework for ultra-deep REY mud lifting.

First-time reviewers should read:

1. `REVIEWER_START_HERE.md`
2. `CLAIM_BOUNDARY.md`
3. `FIELD_VALUE_en.md` or `FIELD_VALUE_ja.md`
4. `NEXT_VALIDATION_PLAN.md`
5. `docs/technical_visual_orientation/index.html`

The repository should be evaluated as a paper companion and validation-planning archive, not as final riser hardware, certified offshore equipment, or a commercial deployment package.
```
