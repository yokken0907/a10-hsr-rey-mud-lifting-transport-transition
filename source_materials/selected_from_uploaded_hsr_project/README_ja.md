# selected_from_uploaded_hsr_project

このフォルダは、アップロードされた `hsr_simulation※レアアース.zip` から、GitHub公開に適した素材だけを抽出したsource snapshotである。

## 含めたもの

- HSR関連 Python scripts
- JSON / logs / small text
- PNG figures
- closure_table.csv

## 除外したもの

- `venv/`
- `site-packages/`
- `__pycache__/`
- `*.pyc`
- compiled binaries
- monolithic code dump

## 注意

これはpaper companion archiveであり、仮想環境ごと公開する完全実行環境ではない。
